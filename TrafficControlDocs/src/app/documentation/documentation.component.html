Cloning - Building - Executing

Installing Git

Windows

Go to https://git-scm.com/ to download the Git-installer. Execute and install.

Ubuntu and devirates

Open a terminal and execute the following commands:
sudo apt update
sudo apt upgrade -yf
sudo apt install git

Clone

Open a terminal and execute the following command:
git clone https://github.com/meik99/TrafficControl.git

Build

To build the subsystems, go to the directory the clone command has created and execute the following command
mvn clean install package
The execute buildDocker.sh
These actions will first build all subsystems. The builds can be found in the "target" directory of each subsystem. The buildDocker file will then build the docker images in the correct order.

Execute as Docker image

By executing startup.sh all services will be started in docker container.

Execute as JAR-Application

By executing the generated jar-files in the target folder the service started will run on the local machine. The jar can be executed with
java -jar [name-of-jar].jar
In both instances can the services be accessed by the following URLs.
